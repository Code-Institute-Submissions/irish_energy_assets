What is this site about?

This site is a means to explore Irelands electrical generation assets across both the transmission and distribution networks.

As the electrical grid migrates to a more distributed system there is an ever growing need to keep track of the assets on the network. For system operators it is important to be aware of what types of generaters are where, what is the generation share of the different types and what eqiupment on the network is reaching capacity.

How is the electrical grid organised?

The infrastructure of wires and cables around Ireland is divided into the distribution network and the transmission network. Traditionally, the transmission network sends electricity from generating stations to main nodes in the country. The electricity passes into the distribution network at these nodes. The distribution network then sends electricity supply to customers' premises. As more and more smaller generators are connected to the grid (i.e. home PV, single wind turbines etc), the distribution network is moving away from purelly delivering energy from transmission nodes to the consumer and begining to feed energy back into the grid.

EirGrid is the acting TSO (Transmission System Operator) responsible for operating and planning the development of the transmission system. ESB Networks is the acting DSO (Distribution Systmem Operator) responsible for the operation, maintenance and development of the electricity distribution network in Ireland. It is also responsible for maintaining and developing the transmission network.

How do I use the site?
  • To see full list of generator assets on record go to Assets.
  • All available information on each asset can be seen by expanding on individual asset.
  • The asset list can be refined using filter options on the Assets page.
  • Filter has to be reset before runing new filter.

Where did the data come from?

The data shown on this site is a combination of publically available data released by ESB Networks and Eirgrid, as well as addtional data included for a more functional uses of the site.

As some data has been included that is not official data from the system operators, the data on this site should only be used for reference purposes.
